 ## join
 
 -join is used to combine rows from tow or more tables based  on a related column betweeen them
 -join help in fetching relateed data from multiple tables in a single query

 <!-- like set theory  -->

Inner join : fetches only mathing rows
Left join  : fetches all rows from left table+matching rows from right table
Right join : fetches all rows from right table + matching rows from left table
Full join  : fetches all rows from both tables, matching where possible
self join  : A table joins with itself
